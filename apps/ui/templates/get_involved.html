We work with welfare institutions, therapy centers, and special-education schools to design and evaluate A3CP in practical, ethically governed pilots. The aim is careful iteration: defining realistic scenarios, testing interaction routines, and evaluating whether the system supports communication without disrupting care.

Partner organizations receive early access to prototypes, structured onboarding, and a direct role in shaping workflows and success criteria. Feedback is translated into concrete changes to interaction flows, schemas, and evaluation.

We take an ethics-first approach with clear privacy boundaries and minimal data retention, avoiding unnecessary capture of raw personal data.

{% set tc_eyebrow = "Contributors" %} {% set tc_title = "For Developers & Researchers" %} {% set tc_lead = "Contribute to an open-source, schema-first platform for adaptive, multimodal communication." %} {% set tc_body %}

A3CP is developed in the open under the GestureLabs GitHub organization. We welcome contributions across backend services, inference pipelines, evaluation tooling, and interface work, with an emphasis on clarity, traceability, and reproducibility.

The system is built around explicit data contracts (schemas) and structured logs so that behavior can be inspected, tested, and improved over time. This makes A3CP suitable both as a practical assistive-technology project and as a research testbed for uncertainty-driven interaction and caregiver-in-the-loop correction.
